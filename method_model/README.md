# Method Model

This folder contains the method-side MATLAB scripts copied from the original `code for method` directory and reorganized for a cleaner repository layout.

## Folder Layout

- `src/`
  - MATLAB scripts and helper functions included in the current workspace
- `source_data/`
  - placeholder directory for required method-model inputs
- `result_data/`
  - placeholder directory for outputs generated by the method-model scripts
- `legacy/`
  - preserved copy of the original English text README from the source folder

## Included Scripts

- `main_calculate_railway_resilience_under_typhoon.m`
- `rail_city2city_time_given_disruption_impact.m`
- `rail_city2city_time_given_disruption_impact_partial_outage.m`
- `city2city_fastest_travel_time_impact.m`
- `run_method_model_task.m`

## What Was Standardized

- the main script now resolves `source_data/` and `result_data/` from the package root instead of assuming they sit beside the script file
- the main script now preserves `fun_type` and throws an explicit error if it is not set
- `run_method_model_task.m` provides a cleaner entry point than relying on a preexisting base-workspace variable

## How To Run

From MATLAB:

```matlab
cd('method_model/src')
run_method_model_task(201)
```

This wrapper sets `fun_type` and runs the original main script.
